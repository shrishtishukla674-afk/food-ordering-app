# food-ordering-app
FoodIt is a beginner-friendly food ordering web application developed as part of my web development learning journey. It allows users to browse restaurants, view menus, add items to cart, and simulate a checkout process.

##  Features
-  Browse list of restaurants
-  View restaurant menus
-  Add items to cart
-  Checkout simulation
-  Responsive design for different screen sizes

##  Tech Stack
- HTML
- CSS
- JavaScript

##  Project Structure
- Home Page → Restaurant Listing
- Restaurant Page → Menu Items
- Cart System → Add/Remove Items
- Checkout Page → Order Summary

##  Purpose
This project was built to strengthen my understanding of:
- Frontend development
- UI/UX design
- DOM manipulation
- Basic web application flow

##  Future Improvements
- User authentication (Login/Signup)
- Payment gateway integration
- Backend integration (Node.js / Database)
- Real-time order tracking
